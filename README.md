## Install
1. On macOS first install the macOS command line tools
    ```sh
    xcode-select --install
    ```

2. Clone the source
    ```sh
    git clone https://github.com/Platinum-Phoenix/.dotfiles ~/.dotfiles
    ```

3. Run the install script
    ```sh
    ./install
    ```
